# Neo Journal 主题制作说明

这份文件保留给主题作者使用，不出现在公开博客主导航里。

更完整的项目级主题规范见：

```text
docs/theme-authoring.md
```

主题制作时请遵守这些硬约束：

- 主题只依赖公开模板上下文：`site`、`theme`、`navigation`、`posts`、`pages`、`collections`、`slot`。
- 公开页面只展示博客内容，不把主题制作规范、项目说明、开发者文档放进主站导航。
- 账号、安装向导、评论等通用能力由 CMS 后端 API 提供，主题只负责入口和视觉。
- 后台可修改站点名称、简介、主题配置、用户和内容；主题不要写死这些数据。
- 默认署名使用 `site.author.name`，当前默认是 `by 伊甸黎明`。
